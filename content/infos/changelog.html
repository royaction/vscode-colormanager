Whats new in Color Manager?

v0.6.2

  • You can now customize the width of the user-interface. Run command "color manager help" for more infos!

  • Fixed error in Command "find colors in selection"

v0.6.1

  • Select multiple colors with CRTL or SHIFT:

    This should be helpful for the operations in the extensions own context menu (f.e. if you want to copy multiple colors from one palette to another).

  • New Command "find colors in selection"

    Use this command to find color values within selections and add them to a palette. Duplicate values will be added only once!

  • 3 new palets with around 3500 colors

    • Library - Wikipedia List of Colors
    • Library - github colornames bestof
    • lospec.com Pixelart Palettes
    Run command "restore factory palettes" to get them! If you made changes to an existing factory palette, do a backup before you run the command!

  • New Command "color manager help"

  • New Command "color manager changelog"

  • Lots of fixes
